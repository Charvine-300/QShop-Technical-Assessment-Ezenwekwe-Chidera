# Shopper-eCommerce-Web-App
This is a clone of the Shopper E-Commerce Web Application.  
The solution to a technical assessment given by QShop Tech inc.

Technologies include: 
- Vite Frontend Tool
- React.js Framework
- Bulma CSS Framework
- Animate.css Animation library
- Axios Client
- Redux Toolkit
- FontAwesome Icons
- React Router DOM
